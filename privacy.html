Privacidad de Datos

Tus datos son tuyos. Kisaki opera bajo una política de "Mínimo Conocimiento Necesario".

1. Recolección de Datos

Solo almacenamos lo estrictamente necesario para que el bot funcione:

  • • Telegram ID: Para identificar tu cuenta y permisos.
  • • Configuraciones: Las reglas que defines para tus canales.
  • • Logs de Actividad: Historial de acciones administrativas (se borran cada 30 días).

2. Almacenamiento Seguro

Toda la información reside en bases de datos encriptadas (Supabase) con seguridad RLS (Row Level Security). Nadie, excepto tú y el sistema automatizado, tiene acceso de lectura a tus configuraciones privadas.
