Política de Privacidad

Protección de datos y seguridad

En Kisaki Manager valoramos tu confianza. Procesamos únicamente la información estrictamente necesaria para el funcionamiento técnico del sistema.

Datos tratados

Recopilamos los siguientes datos mínimos para operar el bot:

  • ID de usuario: Identificador único de Telegram para gestionar tus permisos.
  • Configuraciones de canal: Preferencias que guardas para tus grupos o canales.
  • Registros técnicos: Logs de errores para mantenimiento y depuración.

Seguridad

La seguridad es prioridad. No se almacenan contraseñas, claves privadas de billeteras, ni información financiera sensible en nuestros servidores.

Derechos del usuario

El usuario mantiene el control. Puedes solicitar información sobre tus datos almacenados o la eliminación de registros técnicos cuando sea aplicable contactando al soporte.
